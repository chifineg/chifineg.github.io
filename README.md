## About

This website serves as my academic portfolio, showcasing my background, research interests, engineering projects, and professional experience in Biomedical Engineering.

My research focuses on developing intelligent healthcare technologies that combine biomedical engineering, artificial intelligence, wearable sensing, and rehabilitation engineering to improve mobility, prevent falls, and support healthy ageing.

The website is intended for:

- Prospective research supervisors
- Scholarship committees
- Academic collaborators
- Industry researchers
- Healthcare technology professionals
- Students interested in biomedical engineering

## Featured Projects

### SmartAssist Intelligence (SAI)

An Edge AI-powered intelligent mobility assistance system designed to predict fall risk, support safer mobility, and promote independent living among older adults.

### Robotic Nursing Bed with Speech Recognition

An undergraduate biomedical engineering capstone project involving the design and development of a prototype robotic nursing bed capable of responding to voice commands for patient positioning and movement assistance.

## Running Locally

Clone the repository

```bash
git clone https://github.com/chifineg/chifineg.github.io.git
```

Open the project in Visual Studio Code.

If using Live Server:

1. Install the Live Server extension.
2. Open `index.html`.
3. Click **Go Live**.

Alternatively, simply open `index.html` in your browser.
